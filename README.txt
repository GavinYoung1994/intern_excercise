1. createTable.sql, populateperson.sql, and populateTrans.sql are used to create and populate "person" table and "trans" table according to the data in the dataset_a.csv and dataset_b.csv.

2. autoCreateInsert.hs is a haskell file that helps me convert the data in the csv files into SQL insert statements.

3. dataDump.txt contains all data in the mySQL database.

4. app.js is the file to run the application. Simply enter "node app.js" in the terminal under the correct directory.

5. views folder has all the jade files for frontend display.

6. routes folder has all the files for backend routing and query functions.
